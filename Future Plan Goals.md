# 🔮 Torrnado — Future Plan & Goals

This document outlines the roadmap and future development goals for the **Torrnado** project.

---

## 🚀 Key Objectives

### 1. Make the Torrent Downloader Functional
- Implement native/in-app torrent downloading or integrate deeper with clients.
- Add background download service with notifications.
- Enable speed limiting, queue management, and sequential downloading support.

### 2. Make the Explore Section Work
- Design and integrate the Explore tab.
- Feature trending search terms, top weekly files, and category filters.
- Provide curated recommendations and indexer-specific category views.

### 3. Add Proper Torrent Exploration (IMDb & AniList Integration)
- Integrate with **IMDb** API (or TMDB) for movies/TV shows metadata, ratings, cast, and high-resolution posters.
- Integrate with **AniList** API for anime metadata, seasonal schedules, character details, and posters.
- Map media details directly to search index queries to easily find high-quality torrent releases for a specific movie, show, or anime episode.

### 4. Expand Customization
- Add more custom themes, material palettes, and font controls.
- Support layout toggles (grid vs. list card configurations).
- Enhance widget customizations for home screens.

### 5. Optimization & Design Polish
- Resolve alignment, layout, and UI padding issues across all screens.
- Enhance transitions and visual animations for bottom navigation and search expansions.
- Optimize network caching, Room DB queries, and image loading pipelines.
